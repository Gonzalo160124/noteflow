# Gestión del proyecto – NoteFlow

## Tablero Trello
https://trello.com/b/C4dXJ5v8/noteflow

## Metodología
El proyecto se gestiona con un tablero Kanban en Trello con las siguientes columnas:

- **Backlog**: funcionalidades pendientes de planificar
- **Todo**: tareas listas para empezar
- **In Progress**: tareas en desarrollo activo
- **Review**: tareas terminadas pendientes de revisión
- **Done**: tareas completadas

## Criterio de movimiento de tarjetas
Cada tarjeta representa una funcionalidad principal y se divide en subtareas técnicas concretas. Una tarjeta pasa a Done solo cuando el código está subido a GitHub y funciona correctamente.
