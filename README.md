# NoteFlow

App de productividad móvil desarrollada con React Native y Expo. Permite gestionar tres tipos de contenido: notas de texto, listas de tareas e ideas con etiquetas.

## Stack tecnológico
- React Native + Expo SDK 56
- TypeScript
- Expo Router
- Zustand
- AsyncStorage
- FlashList
- Zod
